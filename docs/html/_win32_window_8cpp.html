◆ MAX_WINDOW_COUNT

#define MAX_WINDOW_COUNT   16

The maximum number of windows that are being stored.

This is used with windowHandles to be able to retrieve a Window from an HWND.

This can be overriden by defining MAX_WINDOW_COUNT in the pre processor before building the DLL.
