◆ computeClockCyclesPerTime()

TAU_DLL void computeClockCyclesPerTime ( const u64  timeoutMS = 50)
noexcept

Used to recompute the number of clock cycles over a given period of time. The time frame over which this calculation is taken is passed by the parameter timeoutMS. If this is taking too long to compute lower the value. If lowering the value causes it to return odd results, increase the value.

Parameters
[in]timeoutMSThe number of milliseconds to timeout for.
